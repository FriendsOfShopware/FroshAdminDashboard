This plugin replaces the Shopware 6 administration dashboard with a modular, customizable widget board. Every user arranges their own dashboard via drag & drop, and the layout is stored individually per user.

The current feature set consists of:

*   Modular widget board
    *   Drag & drop to rearrange widgets
    *   Add, remove and resize widgets in an edit mode
    *   Per-widget settings via a generic settings dialog
    *   Layout is saved per user
*   Analytics widgets (12 charts)
    *   Total sales, number of orders, average order value
    *   New customers and total customers over time
    *   Best-selling products, top manufacturers and orders by country
    *   Payment methods, shipping methods, sales by channel and top promotion codes
    *   Each chart with its own time range and sales-channel filter
*   Operational widgets
    *   Recent orders and recent registrations
    *   B2B customer group requests with inline accept / decline
    *   Pending product reviews with inline approve / decline
*   Productivity widgets
    *   Quick links to the most used areas
    *   Personal notepad
    *   Personal task list
*   Permission aware
    *   Widgets you lack access to are shown greyed-out with an explanation
*   Extensible
    *   Other plugins can register their own widgets and widget groups
